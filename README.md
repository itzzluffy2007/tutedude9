# Task 9: Interactive Transform Button Layout

## Description
This task builds on the responsive laundry services landing section and adds CSS transform interaction for the call-to-action button.

## Features
- Desktop navigation with logo, links, and username element
- Hamburger menu fallback for smaller screens
- Hero section with service copy and image
- Button transform animation on hover
- Responsive layout using media queries

## Files
- `index.html` - page structure with navigation and hero content
- `style.css` - layout styling, responsive menu, and button transform effect

## Usage
1. Open `index.html` in a browser.
2. Hover over the `Book a service today` button to see the transform effect.
3. Resize the browser window to see responsive menu changes.

## Notes
- The hover effect on the button adds interactivity with scale and rotation.
- The mobile menu uses CSS focus behavior for toggling the dropdown.
